Это мои конфигурационные файлы для Void Linux.

Я использовал такие программы:

- **Waybar** — статус-бар
    
- **Fuzzel** — лаунчер приложений
    
- **Alacritty / Kitty** — эмуляторы терминала
    
- **Mako** — демон уведомлений
    
- **swaybg** — установка обоев
    

## Установка

1. Установите перечисленные выше программы и **GNU Stow**.
    
2. Склонируйте репозиторий на свой ПК:
    
    git clone https://github.com/Svetosar/mydotfiles.git
    
3. Перейдите в папку репозитория и выполните команду, чтобы создать симлинки:
    
    cd mydotfiles
    stow .
    
    Либо разложите файлы по нужным папкам вручную.
